# SafeTour: Tourist Safety System
### Smart Tourism Safety Monitoring Platform

---

## 🎯 Main Objectives

1. **Enhanced Tourist Safety** - 24/7 real-time monitoring with panic button and geo-fencing alerts
2. **Digital Identity Management** - Blockchain-secured IDs with QR code verification
3. **Real-Time Incident Response** - 5-minute average response time with live tracking
4. **AI-Powered Monitoring** - Machine learning for anomaly detection and threat assessment
5. **Administrative Control** - Comprehensive dashboards for police and tourism departments
6. **Data Privacy & Compliance** - End-to-end encryption with GDPR compliance

---

## 🛠️ Technology Stack

### **Frontend**
- **Next.js 14** - React framework with server-side rendering
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Modern, responsive UI styling
- **Radix UI** - Accessible component library
- **Recharts** - Data visualization
- **jsPDF** - Digital ID generation

### **Backend**
- **Node.js + Express.js 5** - RESTful API server
- **MongoDB + Mongoose** - NoSQL database with ODM
- **JWT + bcrypt** - Secure authentication
- **Multer** - File upload handling
- **CORS** - Cross-origin support

### **Architecture**
```
Frontend (Next.js) → Backend (Express) → Database (MongoDB)
```
